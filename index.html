Privacy Policy

OAuth Connections Viewer is a Chrome extension that helps users view their OAuth-connected platforms in one place.

  1. Data Collection Scope
    This extension collects OAuth connection info (platform, app name, status) from public DOM elements when explicitly triggered by the user.
  2. Data Storage
    All data is stored in chrome.storage.local within the user's browser and never sent externally.
  3. No External Sharing
    No data is ever shared with any server or third party.
  4. No Automatic Collection
    The extension only runs when the user clicks a button â€” there is no background tracking.
  5. Security
    All processing is done locally within the browser with no outside communication.
  6. User Control
    Users can delete or refresh their stored data at any time.
  7. Cookies and Tracking
    This extension does not use cookies or tracking technologies. All data is stored locally only.
  8. Policy Change Notification
    When the privacy policy changes, users will be notified in advance through extension updates.
