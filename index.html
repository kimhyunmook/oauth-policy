개인정보 보호정책

OAuth Connections Viewer는 사용자가 로그인한 여러 플랫폼의 OAuth 연결 상태를 한눈에 확인할 수 있도록 도와주는 Chrome 확장 프로그램입니다.

  1. 데이터 수집 범위
    이 확장 프로그램은 사용자가 명시적으로 요청할 때, 웹 페이지의 공개된 DOM 요소를 기반으로 OAuth 연결 정보를 수집합니다. 수집된 데이터는 플랫폼 이름, 앱 이름, 연결 상태 등 비민감 정보에 한정됩니다.
  2. 데이터 저장 방식
    수집된 데이터는 사용자의 브라우저의 chrome.storage.local에만 저장되며, 외부 서버로 전송되지 않습니다.
  3. 외부 전송 및 공유 없음
    어떠한 정보도 외부 서버나 제3자와 공유되지 않습니다.
  4. 자동 수집 없음
    확장 프로그램은 자동으로 데이터를 수집하지 않으며, 사용자가 버튼을 눌렀을 때에만 작동합니다.
  5. 보안
    모든 작업은 브라우저 내에서 로컬로 이루어지며 외부 네트워크 통신은 발생하지 않습니다.
  6. 사용자 제어
    사용자는 언제든지 저장된 데이터를 삭제하거나 다시 수집할 수 있습니다.

Privacy Policy

OAuth Connections Viewer is a Chrome extension that helps users view their OAuth-connected platforms in one place.

  1. Data Collection Scope
    This extension collects OAuth connection info (platform, app name, status) from public DOM elements when explicitly triggered by the user.
  2. Data Storage
    All data is stored in chrome.storage.local within the user's browser and never sent externally.
  3. No External Sharing
    No data is ever shared with any server or third party.
  4. No Automatic Collection
    The extension only runs when the user clicks a button — there is no background tracking.
  5. Security
    All processing is done locally within the browser with no outside communication.
  6. User Control
    Users can delete or refresh their stored data at any time.
